# 更新日志

## 2026-07-07 修复

### 1. 富文本空格显示问题
**问题**：段落开头的空格和多个连续空格在富文本内容中不显示
**修复**：在 `app/globals.css` 中为 `.rich-text-content p` 添加了 `white-space: pre-wrap` 和 `word-break: break-word` 样式，确保空格和连续空格正确显示

### 2. 政策管理后台下载列恢复
**问题**：管理后台政策管理页面缺少"下载"列，管理员无法控制单个政策的附件下载开关
**修复**：在 `app/admin/policies/page.tsx` 的政策列表中添加了"下载"列，每行显示下载图标，管理员可以点击切换该政策是否启用附件下载

### 3. 前端政策附件下载显示
**问题**：新增政策添加了附件下载，但在前端政策详情页只能看到政策文本，看不到附件
**修复**：在 `app/policies/[id]/page.tsx` 中简化了下载显示逻辑，移除了对全局配置的依赖，现在只要政策本身启用了下载（`enableDownload: true`）且有附件，就会显示附件下载区域

### 4. 隐藏系统管理的内容审核功能
**问题**：系统管理中的内容审核功能暂时不需要
**修复**：在 `app/admin/settings/basic-settings-tab.tsx` 中隐藏了内容审核设置卡片（代码已注释保留，需要时可快速恢复）

### 5. 之前的修复（持续有效）
- 工会概况页面富文本显示正常，段落间隔正确
- 顶部横幅显示后台配置的自定义标题
- 导航栏固定显示"西安高新区总工会"
- 子页面标题动态获取（不再缓存）

## 部署说明

1. 备份当前部署目录
2. 解压新的部署包
3. 安装依赖：`npm install`
4. 构建项目：`npm run build`
5. 重启服务：`pm2 restart union-portal`
6. 重载 Nginx：`sudo nginx -s reload`

## 技术栈

- Next.js 14.2.35
- Prisma ORM
- MySQL 数据库
- PM2 进程管理
- Nginx 反向代理
